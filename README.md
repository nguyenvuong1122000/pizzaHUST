## Getting started
1. Tải về và cài đặt nodejs tại [đây](https://nodejs.org/dist/v16.13.2/node-v16.13.2-x64.msi).\
   Kiểm tra cài đặt thành công nodejs, npm, npx:
   >node -v
   
   >npm -v

2. Cài đặt thư viện cho project:
    > cd pizzaHUST
 - C1: Sử dụng npm
    >npm install
 - C2: Sử dụng yarn\
   Cài đặt yarn:
   >npm install yarn

   Cài đặt thư viện:
   >yarn
3. Khởi động ứng dụng
   > npm start

    or
   > yarn start
